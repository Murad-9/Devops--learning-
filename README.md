#  linux Bandit (Over The Wire)
This repository contains my solutions for **Bandit Levels 1-20** from OverTheWire.
Bandit is a linux wargame designed to build strong command-line fundamentals through practical challenges.

-----

## What's inside
- Bandit levels 1-20 solutions
- Commands used for each level
- Brief explanations of why each solution works
- Screenshots showing successful completion

-----

## Skills practiced
- Linux file system navigation
- Permissions and ownership
- Searching and filtering files
- Text Processing tools (grep,sort,uniq,tr)
- Networking Basics (nc,openssl)
- SSH and key based authentication
