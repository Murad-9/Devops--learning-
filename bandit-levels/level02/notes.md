## Goal
The password for the next level is stored in a file called 
--spaces in this filename-- located in the home directory

## Solution
ls
cat ./"--spaces in this filename--"

## Explanation
The filename contains spaces and special characters. Using quotes 
around the filename tells the shell to treat it as a single file.

MNk8KNH3Usiio41PRUEoDFPqfxLPlSmx

## Key Lesson
Use quotes for filenames containing spaces.
